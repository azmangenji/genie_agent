# Lines starting with # are comments
# Each line is a regex pattern to match against error lines
# If an error line matches any pattern, it will be filtered (counted as rsmu/dft)
#
# Format:
#   [general]     - Patterns that apply to ALL projects (RSMU/DFT related)
#   [<ip_name>]   - Patterns that only apply to specific IP (e.g., [umc9_3], [umc17_0])
#
# The script will apply:
#   1. All patterns under [general]
#   2. Patterns under the matching [ip_name] section
